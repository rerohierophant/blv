# 交互方式
点赞

评论：语音输入，单击开始单击结束

点击单张图片：多轮对话(录音，点一下开始点一下结束)

双击单张图片：返回得到图片描述

三击单张图片：进入sam（可能需要关闭旁白模式）

点击openai：多轮对话(录音，点一下开始点一下结束)

双击openai：打印并朗读这条朋友圈的描述

# SAM
第一层：得到图片元素并播报；页面上显示整张图可交互；单击对对应物体进行识别播报

下滑切换到第二层：可交互的重要元素，遮住不可交互的元素，

上下滑动切换一二层，左右滑动切换元素。
